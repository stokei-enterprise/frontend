## Frontends

##### Description

Monorepo com os *frontends* da [Stokei](https://stokei.com).

##### Installation

```bash
$ yarn install
```

##### Running the app

```bash
# development
$ yarn dev

# lint code
$ yarn lint

# build production code
$ yarn build

# production mode
$ yarn start
```
